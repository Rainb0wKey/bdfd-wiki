# $divide
Divides the provided numbers.

## Syntax
```
$divide[Numbers]
```

### Parameters
- `Numbers` `(Type: Integer, Float || Flag: Required)`: The numbers to divide. Separate numbers using `;`.

## Example
```
$nomention
Result: $divide[10;$message]
```
