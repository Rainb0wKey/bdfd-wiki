# $botID
Returns the bot's ID.

## Syntax
```
$botID
```

## Example
```
$nomention
My ID is: $botID
```

```discord yaml
- user_id: 803569638084313098
  username: RainbowKey
  color: "#E67E22"
  content: |
    !example
- user_id: 1009018156494368798
  username: BDFD Support
  color: "#378afa"
  bot: true
  verified: true
  content: |
    My ID: 1009018156494368798
```
