# $randomString
Generates a random combination of letters/numbers.

## Syntax
```
$randomString[Length]
```

### Parameters
- `Length` `(Type: Integer || Flag: Required)`: How long the string should be (maximum is `10`).

## Example
```
$nomention
Random String: `$randomString[5]`
```
