# $toLowercase
Converts the provided text to lowercase format.

## Syntax
```
$toLowercase[Text]
```

### Parameters
- `Text` `(Type: String || Flag: Emptiable)`: The text to convert.

## Example
```
$nomention
$toLowercase[$message]
```
