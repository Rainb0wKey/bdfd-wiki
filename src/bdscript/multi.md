# $multi
Muliplies the provided numbers.

## Syntax
```
$multi[Number;..]
```

### Parameters 
- `Number` `(Type: Float, Integer || Flag: Required)`: The numbers to multiply. Use semicolons `;` as a separator to separate multiple numbers.

## Example
```
$nomention
$multi[5;2;9]
```

## Example
```
$nomention
```
