# $botTyping
This function tells Discord that the bot is typing.

## Syntax
```
$botTyping
```

## Example
```
$nomention
$botTyping
Hello $username!
```

- Command started:

``` discord yaml
- user_id: 803569638084313098
  username: RainbowKey
  color: "#E67E22"
  content: |
    !example
```

- Command completed:

``` discord yaml
- user_id: 803569638084313098
  username: RainbowKey
  color: "#E67E22"
  content: |
    !example
- user_id: 1009018156494368798
  username: BDFD Support
  color: "#378afa"
  bot: true
  verified: true
  content: |
    Hello RainbowKey!
```
