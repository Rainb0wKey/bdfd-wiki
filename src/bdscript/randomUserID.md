# $randomUserID
Returns a random user ID from the current server.

## Syntax
```
$randomUserID
```

## Example
```
$nomention
Random User ID: $randomUserID
```

``` discord yaml
- user_id: 803569638084313098
  username: RainbowKey
  color: "#E67E22"
  content: |
    !example
- username: BDFD Support
  color: "#378afa"
  bot: true
  verified: true
  content: |
    Random User ID: 154148273307910144
```
