# $ban[]
Bans the mentioned user with a reason.

## Syntax
```
$ban[Reason]
```

### Parameters
- `Reason` `(Type: String || Flag: Emptiable)`: The reason for the ban, which will be saved in the audit-log.

## Example
```
$nomention
$ban[$noMentionMessage]
<@$mentioned[1]> was banned!
```
