# $serverOwner
Returns the ID of the current server's owner.

## Syntax
```
$serverOwner
```

## Example
```
$nomention
This server is owned by <@$serverOwner>!
```

``` discord yaml
- user_id: 803569638084313098
  username: RainbowKey
  color: "#E67E22"
  content: |
    !example
- username: BDFD Support
  user_id: 1009018156494368798
  color: "#378afa"
  bot: true
  verified: true
  highlight: true
  content: |
    This server is owned by <@RainbowKey>!
```
