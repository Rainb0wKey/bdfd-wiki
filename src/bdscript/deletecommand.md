# $deletecommand
Deletes the author's command message.

> The bot must have the `manage_messages` permission.

## Syntax
```
$deletecommand
```

> Sometimes a message may be deleted, but you will still see it. This is a discord bug.

## Example
```
$nomention
$deletecommand
Hello world!
```
