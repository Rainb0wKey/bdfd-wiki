# $serverVerificationLvl
The server's verification level. Returns `None`, `Low`, `Medium`, `High`, or `Very High`.

## Syntax
```
$serverVerificationLvl
```

## Example
```
$nomention
This server's verification level is $serverVerificationLvl
```
