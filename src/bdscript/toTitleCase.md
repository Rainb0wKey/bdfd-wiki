# $toTitleCase
Converts the first letter of each word to uppercase in the provided text.

## Syntax
```
$toTitleCase[Text]
```

### Parameters
- `Text` `(Type: String || Flag: Emptiable)`: The text to convert.

## Example
```
$nomention
$toTitleCase[$message]
```
