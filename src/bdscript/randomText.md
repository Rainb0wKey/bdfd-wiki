# $randomText
Picks one value from the provided values randomly.

## Syntax
```
$randomText[Text;...]
```

### Parameters
- `Text` `(Type: String || Flag: Emptiable)`: The text to choose from. Separate different texts with `;`.

## Example
```
$nomention
$randomText[Hello;Hi;Hey]!
```
